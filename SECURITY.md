# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability in any Liminal Labs repository, please report it responsibly. **Do not open a public issue.**

Email **security@liminallab.io** with:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if applicable)

We will acknowledge receipt within 48 hours and provide a resolution timeline.

## Scope

This policy covers all repositories under the [Liminal-Lab-io](https://github.com/Liminal-Lab-io) organization.

## Supported versions

| Version | Supported |
|---------|-----------|
| Latest release | Yes |
| Previous releases | Best effort |
